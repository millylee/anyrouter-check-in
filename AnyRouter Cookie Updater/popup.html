🔄 AnyRouter Cookie Updater

GitHub 配置
需要 repo 权限（或 Environment secrets 写入权限）
留空则推送到 repository secrets
账号配置
domain: 站点地址 | api_user: 用户ID | env_key_suffix: secret 后缀（生成 ANYROUTER_ACCOUNT_{suffix}）
cookie_name: 可选，默认 "session"
定时设置
建议 360 分钟（6 小时），与签到频率一致
⚡ 工作原理：
1. 从浏览器中提取已登录站点的 session cookie
2. 通过 GitHub API 加密后推送到 Environment Secrets
3. 签到脚本从 ANYROUTER_ACCOUNT_* 读取最新 cookie

⚠️ 前提：你需要在浏览器中保持各站点的登录状态
